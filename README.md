# urlshortener
Url shortener using Asp.Net MVC, Asp.Net Web Api, AngularJs, Sql Server 2014

### Installation
1- Run Sql scripts:
* 01_CrateDB.sql
* 02_CrateLogin.sql
* 03_CrateSqlUser.sql
* 04_CreateTables.sql

2- Open the solution rm.urlshortener in Visual Studio 2015

3- Run the project
